# Full-stec — Практики 1–6 (Frontend + Backend)

В этом репозитории собраны выполненные практические задания по курсу "Фронтенд и бэкенд разработка" (практики 1–6).

Структура (локально):
- `практика 1-6/` — все файлы текущего проекта (HTML/CSS/SASS, сервер Express, тесты, фронтенд).

Быстрый старт (локально):

1) Перейдите в папку проекта:

```bash
cd "практика 1-6"
```

2) Установите зависимости и запустите API:

```bash
npm install
node app.js
# API: http://localhost:3000
# Swagger UI: http://localhost:3000/api-docs
```

3) Запустите статический фронтенд в отдельном терминале (пример):

```bash
python3 -m http.server 3001
# Затем откройте http://localhost:3001/products.html
```

4) Для быстрых тестов API используйте скрипт:

```bash
./test_api.sh
# результаты сохраняются в test_*.json
```

Файлы ключевые:
- `app.js` — backend (Express, CRUD для товаров, Swagger)
- `index.html`, `styles.scss`, `styles.css` — карточка товара (Практика 1)
- `products.html` — простой frontend, который читает API (Практика 4)
- `test_api.sh` — пример автоматизированных curl-тестов (Практика 3)
- `README.md` (этот файл) — инструкция

Где загрузить изображения
- Папка для изображений: `практика 1-6/assets/images/`.
- Назовите файлы так, как указано в `app.js` (пример):
	- `watch.jpg` → "Умные часы"
	- `headphones.jpg` → "Беспроводные наушники"
	- `fitness.jpg` → "Фитнес-браслет"
	- `speaker.jpg` → "Портативная колонка"
	- `ssd.jpg` → "Внешний SSD 512GB"
	- `cable.jpg` → "USB-C кабель"
	- `case.jpg` → "Чехол для телефона"
	- `charger.jpg` → "Зарядное устройство 30W"
	- `keyboard.jpg` → "Клавиатура механическая"
	- `mouse.jpg` → "Мышь беспроводная"

После загрузки картинок перезапустите сервер (`node app.js`) — фронтенд
`products.html` начнёт отображать картинки автоматически.

Если при попытке загрузки на GitHub возникнут ошибки аутентификации, выполните push локально с настроенным SSH-ключом или Personal Access Token (PAT). Я попытался добавить remote и выполнить push — если потребуется, пришлю команды и подсказки по настройке доступа.
